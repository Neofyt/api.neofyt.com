APIs

  • json : returns the list of parameters sent in the query string as a JSON file

    1. http://api.neofyt.com/json/?value=test&version=20.0&callback=foo

  • img : creating image placeholder by providing dimensions, colors and eventually text in the query

    1. http://api.neofyt.com/img/?300.150.558ed5.ffffff.text // Width.Height.Text color.Background color.Text (Text will be image size if parameter left empty)

  • jquery : get a specific version of jQuery by specifiyng the version number or get the last one

    1. http://api.neofyt.com/jquery/?1.8.2 // Uncompressed version
    2. http://api.neofyt.com/jquery/?1.8.2-min // Minified version
    3. http://api.neofyt.com/jquery/?last // Always get the last version
